# Proctor's Shred Journey

Interactive workout tracker. All progress saved to your browser's local storage.

## Files in this repo
- `index.html` — the entire app
- `favicon.png` — browser tab icon
- `apple-touch-icon.png` — icon when saved to iOS home screen
- `og-image.jpg` — preview shown when the link is shared on iMessage, WhatsApp, etc.
- `.nojekyll` — tells GitHub Pages to skip Jekyll processing
